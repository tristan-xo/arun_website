PELTIER COOLER FOR GAMING LAPTOPS

I developed a smart cooling system using a Peltier module, designed to regulate temperature efficiently based on real-time data. Initially, the system used a DHT11 sensor to monitor ambient temperature, but I later upgraded it to read GPU temperature directly from my computer. The Peltier module automatically activates at 55°C and deactivates at 35°C, with a cooling fan running for an additional 5 minutes to dissipate heat. A 16x2 I2C LCD displays temperature readings and system status. To enhance functionality, I am integrating IoT capabilities using ESP8266 for remote monitoring and control via MQTT. This project demonstrates my expertise in automation, embedded systems, and IoT integration.
